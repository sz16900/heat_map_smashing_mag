# Smashing Magazine - Heat Map

> In this project, we are going to take a basic layout of Smashing Magazine and try to re-create a heat map. This project is completed by Seth Zea and Simon Peter Debbarma, in partial requirements of the Microverse cirriculum.

### Prerequisites

- Node.js (for Browersync and Stylelint)
- Web Browser
- CLI
- IDE

### Dependencies

- Browsersync
- Stylelint

### Install

`git clone`

`npm install`

### Run Linter Test

`stylelint "*/*.css"`

> https://stylelint.io/user-guide/cli

### Autofix CSS Linting Errors

`stylelint "*/**.css" --fix`

### Deployment

`npm start`
